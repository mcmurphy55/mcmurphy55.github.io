# iOS Apps by mcmurphy55

GitHub Pages portfolio for three iOS apps:

- Cinemagora
- Attention
- Uchinalingo

## Development

```sh
npm install
npm run dev
```

## Checks

```sh
npm run lint
npm run build
```

## Deployment

This repository is intended to be published at:

```text
https://mcmurphy55.github.io/
```

GitHub Pages deploys from `.github/workflows/deploy.yml` after pushes to `main`.

## Assets

Web-sized copies of app screenshots live under `public/apps/`. Original source assets are kept in each app repository and are not modified by this portfolio site.
